# Media Tag Dictionary

Use lowercase English tags only, comma-separated.

## Global Rules
- Keep tags short and consistent.
- Use 4-7 tags per image.
- Order: `type, shot, category, subgroup, material, optional`.

## 1) Type Tags
- `product`
- `mockup`
- `print-sample`

## 2) Shot Tags
- `cover` (listing/main image)
- `detail` (material/process close-up)
- `lifestyle` (real-world scene)
- `installation` (mounting/usage process)

## 3) Category Tags
- `marketing-business-print`
- `stickers-labels-decals`
- `signs-rigid-boards`
- `banners-displays`
- `windows-walls-floors`
- `vehicle-graphics-fleet`

## 4) Material Tags
- `paper`
- `adhesive`
- `non-adhesive`
- `rigid`
- `hardware`

## 6) Ready-to-Use Examples

- Vinyl banner cover:
  - `product,cover,banners-displays,vinyl-banners,non-adhesive,vinyl`
- Mesh banner detail:
  - `product,detail,banners-displays,mesh-banners,non-adhesive,vinyl`
- Business cards close-up:

## 7) Alt Text Formula
- `Product + key material/finish + usage scene`
- Example: `13oz vinyl banner with grommets installed on storefront`
